# 项目总览

## 1. 项目背景

本项目是宜兴新奇环保数字孪生项目。

当前场景和数据接入已完成。主要问题是呈现效果、基于 UE 动画的演示视频、系统先进性展示与阐述，尚未满足客户营销诉求。

POC 提交时间：2026-04-20，周一。

本文作为 Obsidian 项目入口，用于导航项目背景、课题边界、文档规范和后续架构设计材料。

## 3. 当前课题范围

本阶段关注 POC 交付前的展示效果改善与营销表达补强。

重点方向包括：

- 数字孪生场景呈现效果优化
- 基于 UE 动画的演示视频优化
- 系统先进性展示逻辑梳理
- 面向客户营销诉求的讲解结构设计
- POC 交付材料范围确认
- 后续客户需求补充与归档

## 4. 架构设计关注点

后续技术和展示方案设计需要重点回答以下问题：

- 客户认为当前呈现效果不满足诉求的具体原因是什么？
- POC 视频需要展示哪些关键业务场景？
- 哪些 UE 动画、镜头、转场和特效能直接提升展示说服力？
- 系统先进性应从哪些技术能力和业务价值维度阐述？
- POC 交付材料应包括视频、演示脚本、功能说明还是技术亮点说明？

## 5. 模块接口设计关注点

后续如进入系统深化，再围绕以下对象展开接口和模块设计：

- 孪生体场景构建统一接口：以孪生体为场景搭建最小管理单元，统一身份、层级、空间、数据绑定和表现引用。
- 场景数据接口：明确现有数据接入来源、更新方式和展示字段。
- 展示控制接口：支持镜头路径、动画触发、状态切换和讲解节奏。
- 业务对象接口：明确设备、点位、状态、告警或运营指标的呈现方式。
- POC 演示接口：支撑一键演示、分段播放、镜头复位和演示状态恢复。

## 7. 当前结论

本项目当前不是场景和数据接入问题，而是 POC 展示效果、UE 动画视频表达和系统先进性阐述问题。

后续讨论优先围绕客户营销诉求、展示改善重点、POC 演示脚本、视频方案和系统亮点表达展开。
