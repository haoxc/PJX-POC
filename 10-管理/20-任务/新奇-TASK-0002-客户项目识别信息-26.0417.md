# 新奇 TASK-0002 客户项目识别信息

## 1. 任务目标

在 POC 视频和关键截图中明确体现“宜兴新奇环保”项目识别，避免客户归属只依赖口头讲解。

## 2. 任务范围

| 项目 | 内容 |
| --- | --- |
| 处理对象 | 主厂房、孪生区域入口、核心设备区、视频标题层 |
| 表达内容 | 客户名称、项目名称、LOGO 或文字标题 |
| 使用场景 | 视频开场、主厂房识别段、局部设备收束画面 |
| 输出用途 | 支撑客户归属和营销表达 |

## 3. 输入材料

| 输入 | 来源 |
| --- | --- |
| 客户标识资料 | [[新奇-客户标识]] |
| 整体场景评估 | [[../../02-需求/场景-整体场景(main)/整体场景评估报告-26.0417]] |
| POC 计划 | [[../10-计划/POC可行计划-26.0417]] |

## 4. 输出物

| 输出 | 要求 |
| --- | --- |
| 客户识别方案 | 明确使用 LOGO、项目名或文字标题 |
| 视频画面呈现 | 至少 1 个镜头中清晰出现客户或项目识别 |
| 关键截图 | 至少 1 张能体现客户识别 |
| 脚本文案 | 与画面标识一致，不夸大客户确认内容 |

## 5. 验收标准

| 标准 | 通过条件 |
| --- | --- |
| 客户可识别 | 观众能明确这是宜兴新奇环保项目 |
| 标识清晰 | 字号、位置、对比度满足视频观看 |
| 不遮挡主体 | 标识不遮挡主厂房、工艺区和数据标签 |
| 口径一致 | 视频标题、截图和讲解词使用同一项目名称 |

## 6. 边界说明

本任务只解决客户和项目识别。

不处理以下事项：

- 品牌 VI 重新设计。
- 未确认 LOGO 的正式授权。
- 系统 UI 全局风格设计。
- 客户宣传片完整包装。

## 7. 风险与处理

| 风险 | 处理 |
| --- | --- |
| LOGO 素材未确认 | 先使用项目名称文字标识 |
| 标识影响画面 | 使用短时标题层或固定在主厂房立面 |
| 名称口径不确定 | 使用“宜兴新奇环保数字孪生 POC”作为临时口径并标记待确认 |
