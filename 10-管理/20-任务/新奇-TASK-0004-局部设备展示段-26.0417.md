# 新奇 TASK-0004 局部设备展示段

## 1. 任务目标

完成至少一个核心局部设备段，用于证明系统不是静态场景，而具备设备定位、状态表达和业务说明能力。

## 2. 任务范围

| 项目 | 内容 |
| --- | --- |
| 优先对象 | 厌氧罐区域 |
| 目标时长 | 15-20 秒 |
| 表达内容 | 设备识别、镜头推进、环绕或定帧、状态或数据标签 |
| 输出用途 | 支撑 POC 中“局部设备做亮点”的策略 |

## 3. 输入材料

| 输入 | 来源 |
| --- | --- |
| 厌氧罐局部分析 | [[../../02-需求/局部-厌氧罐/场景分析-厌氧罐-26.0417]] |
| POC 计划 | [[../10-计划/POC可行计划-26.0417]] |
| 客户标识 | [[新奇-客户标识]] |

## 4. 输出物

| 输出 | 要求 |
| --- | --- |
| 局部设备镜头 | 包含推进、环绕或定帧 |
| 设备标签 | 至少标识设备名称或功能 |
| 状态表达 | 至少包含 1 类状态、参数或能力提示 |
| 局部截图 | 至少 1 张可用于汇报材料 |

## 5. 验收标准

| 标准 | 通过条件 |
| --- | --- |
| 设备主体清楚 | 观众能看出当前展示的核心设备 |
| 业务价值清楚 | 能说明该设备与环保处理或运维管理有关 |
| 数据能力可见 | 出现状态、参数、告警或能力提示 |
| 视频可衔接 | 能从整体镜头自然进入该局部段 |

## 6. 边界说明

本任务只完成一个局部设备展示样板。

不处理以下事项：

- 所有设备逐一精修。
- 完整工艺流程仿真。
- 未确认数据的真实联动。
- 复杂管线动画。

## 7. 风险与处理

| 风险 | 处理 |
| --- | --- |
| 厌氧罐细节不足 | 优先用标签、镜头、数据卡片补足表达 |
| 工艺数据未确认 | 使用演示字段并标记待确认 |
| 时间不足 | 保留设备镜头和状态卡片，放弃复杂动画 |
